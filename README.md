# 🏛️ Câmara Municipal de Sinop - Inovação & Tecnologia

Bem-vindo ao GitHub oficial da **Câmara Municipal de Sinop - MT**. Nosso setor de tecnologia atua para modernizar o poder legislativo municipal, construindo ferramentas que promovem transparência, eficiência administrativa e um melhor atendimento ao cidadão sinopense.

## 🚀 Nossa Missão Tecnológica
* **Transparência e Cidadania:** Desenvolvimento de plataformas que facilitam o acesso à informação pública e aos processos legislativos.
* **Modernização Administrativa:** Criação de sistemas internos para a digitalização, otimização de fluxos e controle de dados institucionais.
* **Soluções Customizadas:** Engenharia de software e hardware adaptada às necessidades exclusivas das sessões plenárias e da administração pública.

## 💻 Stack Tecnológica
Nossas soluções são construídas com foco em performance, segurança e escalabilidade, utilizando tecnologias modernas:

* **Frontend:** React, TypeScript
* **Backend:** Go, Python
* **Infraestrutura & Deploy:** Docker, Ubuntu Server
* **Hardware & IoT:** Arduino e ESP32 para automações de sessões e infraestrutura física

## 📌 Principais Iniciativas e Sistemas
* **Sistema de Gestão Legislativa:** Plataforma dedicada à documentação, tramitação e armazenamento seguro de dados institucionais.
* **CâmaraGestão:** Nosso sistema proprietário de helpdesk e gestão de ativos de TI, garantindo o funcionamento ininterrupto da administração.
* **Automação de Plenário:** Soluções customizadas de hardware com relés de segurança para o gerenciamento de canais de áudio e infraestrutura durante as sessões públicas.
